Modifier le nom des chapitres

Parfait ! Maintenant, nous y voyons plus clair. Nous allons maintenant renommer les noms des chapitres, qui ont été générés automatiquement d'après le nom des séquences. Vous suivez ?

Pour cela, deux possibilités.

  • Dans le visualiseur, le label du chapitre est éditable. Vous pouvez le modifier directement en cliquant dessus.
  • Dans le panneau d'édition, les labels sont également éditables.

La modification du label d'un chapitre à travers l'une ou l'autre de ces méthodes entraîne la synchronisation avec les autres apparitions du chapitre en question dans l'interface (ligne de temps, visualiseur, panneau d'édition.

Notez qu'il n'est pas possible de modifier le label d'un chapitre depuis la ligne de temps, qui est seulement destinée à modifier les dates d'occurrence d'un évènement.

Effectuez les modifications suivantes :

  • Renommer "intro-site-du-zero.mp4" vers "Introduction"
  • Supprimer les préfixes "HTML5/CSS3 - " issus des titres des séquences vidéo importées

Pour valider, terminez chaque modification de label en appuyant sur Entrée, ou en appuyant sur "Synchroniser" quand tous les labels auront été édités.
