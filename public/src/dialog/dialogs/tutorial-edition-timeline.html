Dans la ligne de temps

Comme vous pouvez le voir, la ligne de temps contient trois types d'évènements : les séquences vidéo, la table des matières et les repères de chapitrage.

Chacun de ces évènements est manipulable indépendamment, à la fois dans le temps et dans l'espace.

Les séquences vidéo et la table des matières sont visibles dans le visualiseur. Les repères de chapitrage sont également visibles à travers le label du chapitre en cours qui s'affiche dans le visualiseur (en haut à gauche).
