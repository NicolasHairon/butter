Importation des ressources

Les ressources sont en cours d'importation. Si une URL n'est pas conforme, un message d'erreur s'affichera.

Importation ne signifie pas que les ressources sont ajoutées dans la ligne de temps, mais simplement affichées dans la galerie de médias du panneau d'édition. Il y a deux manières d'utiliser en pratique ces ressources dans la ligne de temps.

Le bouton "Ajouter tout" permet d'ajouter les ressources importées de manière séquentielle dans la ligne de temps. Il est aussi possible d'ajouter individuellement chaque ressource en cliquant sur sa vignette.

Nous allons ajouter toutes les ressources séquentiellement. Cliquez sur le bouton "Ajouter tout".
